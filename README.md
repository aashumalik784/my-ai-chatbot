## ✨ Features

### 🎨 **User Interface**
- 🌙 Modern dark theme with purple & blue gradient accents
- 🎭 Glassmorphism effects with backdrop blur
- 📱 Fully responsive mobile design (sidebar collapses on mobile)
- ✨ Smooth animations (fade-in, slide-up, hover effects)
- 🎯 Intuitive chat history management
- ⚙️ Settings panel ready for customization

### 💬 **Chat Capabilities**
- ⚡ Real-time streaming responses token by token
- 🔄 Multiple AI model selection (Llama 3.3, Mixtral, Gemma)
- 💾 Persistent chat history with localStorage
- 🏷️ Auto-saving chat titles from first message
- 🔍 Full conversation context support
- 📊 Token usage tracking

### 💻 **Code & Content**
- 🎨 Markdown rendering support
- 📝 Syntax-highlighted code blocks
- 📋 One-click copy button for code
- **Bold** and *italic* text formatting
- Proper line break handling

### 🔒 **Security & Performance**
- 🔐 Helmet security headers
- 🚫 CORS protection
- ⏱️ Rate limiting (100 requests/hour per IP)
- 🛡️ Input validation & sanitization
- 📦 10MB request size limit
- ✅ Graceful error handling

### 🌐 **Cross-Platform**
- 🖥️ Desktop-optimized interface
- 📱 Mobile-first responsive design
- 🍎 iOS support
- 🤖 Android support
- 🌍 Browser compatibility (Chrome, Firefox, Safari, Edge)

## 🛠️ Tech Stack

### **Frontend**
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables, Flexbox, Grid
- **JavaScript (ES6+)** - Vanilla JS (no frameworks)
- **LocalStorage API** - Client-side persistence
- **Fetch API** - HTTP requests

### **Backend**
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **Axios** - HTTP client
- **Helmet** - Security middleware
- **CORS** - Cross-origin resource sharing
- **Express Rate Limit** - Rate limiting middleware
- **dotenv** - Environment variables

### **AI & APIs**
- **Groq API** - LLM inference
- **Llama 3.3 70B** - Primary model
- **Mixtral 8x7B** - Alternative model
- **Gemma 7B** - Lightweight model

### **Deployment**
- **GitHub Pages** - Frontend hosting
- **Render** - Backend hosting
- **GitHub** - Version control
